# Claude RPG

यह एक सिमुलेशन-आधारित अभियान आरपीजी है, जिसमें क्लाउड कहानी को प्रस्तुत करता है, इंजन सत्य को बनाए रखता है, और दुनियाएं अफवाहों, दबाव, गुटों, रिश्तों, अर्थव्यवस्था और सिस्टम के माध्यम से विकसित होती हैं, जो सार्थक निष्कर्षों की ओर ले जाती हैं। इसे खेलें या इस पर निर्माण करें।

## Claude RPG क्या है?

Claude RPG, [AI RPG इंजन](https://github.com/mcp-tool-shop-org/ai-rpg-engine) के ऊपर बना है - एक नियतात्मक सिमुलेशन रनटाइम जिसमें 29 मॉड्यूल हैं जो युद्ध, अनुभूति, धारणा, गुटों, अफवाहों, विश्वास के स्रोतों, NPC एजेंसी, साथियों, खिलाड़ी के प्रभाव, रणनीतिक मानचित्रों, आइटम पहचान, उपकरण के स्रोतों, उभरती संभावनाओं, अभियान आर्क का पता लगाने और अंतिम खेल ट्रिगर्स को कवर करते हैं। क्लाउड का काम व्याख्या करना, वर्णन करना और बोलना है। इंजन का काम सत्य को बनाए रखना है।

सुनहरा नियम: **क्लाउड प्रस्ताव करता है, इंजन निष्पादित करता है।**

खिलाड़ी स्वतंत्र रूप से टेक्स्ट लिखते हैं। क्लाउड इरादे की व्याख्या करता है, इंजन नियतात्मक रूप से कार्यों को हल करता है, धारणा फ़िल्टर यह तय करते हैं कि खिलाड़ी ने वास्तव में क्या देखा, और फिर क्लाउड केवल उस चरित्र द्वारा महसूस की गई चीज़ का वर्णन करता है - जिसमें आवाज, ध्वनि प्रभाव और इमर्शन रनटाइम द्वारा तैयार किए गए परिवेशी ऑडियो शामिल हैं।

NPC स्क्रिप्ट नहीं पढ़ते हैं। वे विश्वासों, यादों, गुट की वफादारी और अफवाहों से बोलते हैं। वे कारणों से झूठ बोलते हैं। वे कारणों से अनिश्चित होते हैं। वे कारणों से इनकार करते हैं। डायरेक्टर मोड आपको यह देखने की अनुमति देता है कि वास्तव में ऐसा क्यों है।

## अपना बनाएं

Claude RPG सिर्फ एक गेम नहीं है - यह AI RPG इंजन पारिस्थितिकी तंत्र के लिए एक संदर्भ कार्यान्वयन है। इसका उपयोग अपने स्वयं के सिमुलेशन-आधारित कथा अनुभवों के लिए शुरुआती बिंदु के रूप में करें।

| क्या आप चाहते हैं... | उपयोग करें |
|------------|-----|
| **Play right now** | `npx claude-rpg play` (इंटरैक्टिव दुनिया और चरित्र चयन) |
| **Create a new world** | `npx claude-rpg new "your world concept"` |
| **Author worlds visually** | [वर्ल्ड फोर्ज](https://github.com/mcp-tool-shop-org/world-forge) - 2D रचना स्टूडियो जिसमें मानचित्र संपादक, NPC बिल्डर और सत्यापन शामिल हैं। |
| **Validate world data** | [कैनन आर्काइव](https://github.com/mcp-tool-shop-org/cannon-archive) - स्कीमा सत्यापन, स्टोरीबोर्ड परीक्षण, निर्यात पाइपलाइन। |
| **Build a custom runtime** | [@ai-rpg-engine](https://github.com/mcp-tool-shop-org/ai-rpg-engine) पैकेज सीधे आयात करें - क्लाउड को किसी भी LLM से बदलें, अपना UI जोड़ें। |
| **Add new game modules** | इंजन को फोर्क करें, समाधान पाइपलाइन में मॉड्यूल जोड़ें और उन्हें पंजीकृत करें। |

इंजन LLM-अज्ञेयवादी है। Claude RPG एंथ्रोपिक मॉडल का उपयोग करता है, लेकिन मुख्य इंजन में LLM की कोई निर्भरता नहीं है - आप इसे किसी भी मॉडल से जोड़ सकते हैं या पूरी तरह से नियतात्मक रूप से बिना वर्णन के चला सकते हैं।

## इंस्टॉल करें

```bash
npm install claude-rpg
```

या सीधे चलाएं:

```bash
npx claude-rpg play
```

## शुरुआत कैसे करें

```bash
# Play — interactive world and character selection
npx claude-rpg play

# Accelerated campaign pacing
npx claude-rpg play --fast

# Generate a new world from a prompt
npx claude-rpg new "A flooded gothic trade city ruled by three merchant houses"

# Use the engine in your own project
npm install @ai-rpg-engine/core @ai-rpg-engine/modules
```

अपना एंथ्रोपिक एपीआई कुंजी सेट करें (केवल क्लाउड वर्णन के लिए आवश्यक):

```bash
export ANTHROPIC_API_KEY=sk-ant-...
```

## v1.5.0 में क्या नया है

v1.5 एक मजबूत संस्करण है — 67 बग ठीक किए गए हैं, 22 सुरक्षात्मक उपाय किए गए हैं, और तीन नई सुविधाएँ हैं जो गेम को जीवंत महसूस कराती हैं।

| सुविधा | इसका क्या मतलब है |
|---------|--------------|

| **Terminal colors + spinner** | क्षति, उपचार और एनपीसी (NPC) नामों के लिए एएनएसआई रंग। एलएलएम (LLM) कॉल के दौरान एनिमेटेड स्पिनर। |

## आर्किटेक्चर।

```
Player types freeform text
    |
[1] ACTION INTERPRETATION (Claude)
    Input: player text + verbs + entities + exits
    Output: { verb, targetIds, confidence }
    |
[2] ENGINE RESOLUTION (deterministic)
    engine.submitAction() -> ResolvedEvent[]
    |
[3] PERCEPTION FILTERING (deterministic)
    presentForObserver() -> what the player saw
    |
[4] HOOKS: pre-narration
    Zone ambient, combat alerts, death effects
    |
[5] NARRATION PLAN (Claude)
    Input: filtered scene + presentation state
    Output: NarrationPlan { text, sfx, ambient, music, UI }
    |
[6] AUDIO DIRECTOR
    Priority, ducking, cooldowns -> AudioCommand[]
    |
[7] PRESENTATION
    Voice synthesis + SFX + ambient via voice-soundboard
    Text rendering to terminal
    |
[8] NPC DIALOGUE (Claude, if speaking)
    Grounded in cognition: beliefs, memories, faction, rumors
    Voice-cast per NPC
```

## इमर्शन रनटाइम (संस्करण 0.2)
